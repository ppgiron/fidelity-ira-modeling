# Shared Libraries

This directory contains reusable utility functions and services used across the application.

## Structure

- `db.ts` - Dexie.js database configuration
- `encryption.ts` - Web Crypto API wrapper for encryption/decryption
- `formatters.ts` - Utility functions for formatting data
- `validators.ts` - Common validation utilities
- `error-handling.ts` - Centralized error handling
